# Contributing to Awesome Perp DEXs

This space ships fast and the list only stays useful with community help. Thanks for taking the time to contribute.

## Quick rules

- The project must be a **live** (or public-testnet) on-chain venue, tool, or piece of infrastructure that directly serves perp traders.
- **One entry per row**, alphabetical-ish within sub-groupings where possible.
- **Use the existing column structure** of the section you're editing — don't introduce a new format unless you're adding a brand-new section.
- **Use the legend** consistently:
  - 🟢 Live · 🟡 Beta / Testnet · 🔴 Inactive · ⚡ Top-volume · 🆕 Launched 2025/26
- **No referral spam.** One referral link per row at most. Maintainer-default referrals are fine but not required.

## What we accept

- Perp DEXs (CLOB, oracle pool, RFQ, intent-based — all welcome)
- Perp aggregators / routers
- Trading terminals & front-ends for perp DEXs
- Analytics, dashboards, and stats sites
- Whale-tracking & on-chain intel tools
- Copy-trading platforms
- Trading bots (must integrate with at least one perp DEX)
- Mobile apps for perp trading
- Portfolio trackers that surface perp positions
- Block explorers / indexers / RPCs that meaningfully serve perp traders
- Decentralized options & structured-product venues (closely related to perps)
- RWA perps (equities, commodities, rates, real-estate)
- MCPs / AI agents for perp trading

## What we don't accept

- Pure spot DEXs without perpetual futures
- Centralized exchanges (CEXs)
- Whitepapers without working products
- Abandoned / unmaintained projects (we'll mark them 🔴 if they were once active)
- Affiliate-link spam
- Duplicate listings — please grep the README first

## How to submit

1. Fork the repo
2. Create a feature branch (`feature/add-foo-perp`)
3. Edit the relevant section
4. Open a PR with:
   - The project name + URL
   - One-line description
   - Why it belongs in the section you placed it in
   - Optional: any chain / volume / leverage stats you can verify

## Reporting issues

Found a broken link or inactive project? Open an issue (or a PR) with a quick note.

## Updating stats

Stats (volume, OI, etc.) in the top tables come from DefiLlama / PerpVision and drift over time. We refresh them periodically. If you spot a particularly stale number, a PR with the latest is welcome.
